This Directory contains all files related to the attempt to verfiy the AES CTR implementation.

xor.il: The relevant parts of the CTR mode implementation (basically the 16 XOR operations). This is generated by Hamed's Lifter and requires some small modifications done by hand.

xor.smt: The smt formula in smt2lib format. Containing the Pre and Postconditions. Generated by BAP topredicate functionality and can be solved using Z3. 

All other files: Basically test files
